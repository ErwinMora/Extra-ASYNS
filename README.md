Mediante este proyecto se pone en practica lo aprendido en las clases, aplicando modelos de **Machine Learning** bajo lo que es el **Análisis Supervisado** y **Análisis No Supervisado**.

- 3.10.11

## Dependencias
- Pandas
- Numpy
- Sklearn
- Seaborn
- Matplotlib
- Jupyterlab
- Notebook
